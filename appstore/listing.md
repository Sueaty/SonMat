
## App Store Connect 제출 체크리스트

- [ ] Apple Developer Program 계정 활성화 확인
- [ ] App ID 등록 (`com.sueaty.sonmat` 또는 사용 중인 Bundle ID)
- [ ] 앱 아이콘 준비 — 1024×1024 px PNG (알파 채널 없음)
- [ ] 스크린샷 6.9" 필수, 나머지 선택
- [ ] 개인정보 처리방침 URL 호스팅 확인: https://sueaty.github.io/SonMat/privacy/
- [ ] 앱 설명, 키워드, 프로모션 텍스트 입력
- [ ] 개인정보 데이터 관행 입력
  - 수집 데이터: 사용 데이터 (Firebase Analytics — 익명, 앱 기능 개선 목적)
  - 사용자 추적 없음
- [ ] 연령 등급 설문 완료 (4+)
- [ ] 앱 빌드 TestFlight 또는 직접 제출
- [ ] "앱 검토에 제출" 클릭
